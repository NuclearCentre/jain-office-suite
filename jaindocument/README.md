# WordPad Pro
A full-featured Windows word processor built with Electron.

## Features
- Save as .docx, .doc, .html, .txt
- Open .docx, .html, .txt files
- Full ribbon with Home, Insert, Layout, Review, View tabs
- Bold, italic, underline, font family/size/color
- Tables, images, links, special characters
- Find & Replace, spell check, word count
- Page layout, margins, orientation, page size
- Placeholder text (watermark style)
- Save dialog with unsaved changes warning
- GitHub auto-updater + local batch updater

## Build Instructions
1. Install Node.js from https://nodejs.org
2. Run: npm install
3. Run: npm run build-win
4. Installer appears in dist/ folder

## Backup
Run backup-to-github.ps1 after every task to save progress.
