## 8. Close Window Fix

**Problem history:** X button not closing was the most persistent bug across multiple sessions.  
**Root cause:** Original code used IPC round-trip (renderer had to reply to close). If renderer errored, window stayed open forever.  
**Final fix in main.js:**
```javascript
async function handleClose(e) {
  e.preventDefault();
  const choice = await dialog.showMessageBox(mainWindow, {
    type: 'question',
    buttons: ['Close', 'Cancel'],
    defaultId: 0, cancelId: 1,
    title: 'Close JainSheet',
    message: 'Close JainSheet?',
    detail: 'Any unsaved changes will be lost.'
  });
  if (choice.response === 0) {
    mainWindow.removeListener('close', handleClose);
    mainWindow.destroy();
  }
}
mainWindow.on('close', handleClose);
```
**Key:** `removeListener` before `destroy()` prevents re-triggering.  
**Note:** Old `.exe` in `D:\Jain Sheet EXE Folder\` must be uninstalled — shortcut may still point there.

## 9. Grid Visibility Fix

**Problem:** Grid area showed blank white space — no cells/lines visible.  
**Root cause:** `grid-container` div had `display:block` inline style overriding `flex:1` from CSS, causing zero height in Electron's Chromium.  
**Fix:**
- Removed all inline styles from `grid-container` div in HTML
- CSS: `#grid-container { flex: 1 1 0 !important; overflow: auto; position: relative; min-height: 0; min-width: 0; }`
- CSS: `#app > * { flex-shrink: 0 }` + `#grid-container { flex-shrink: 1 !important }`
- JS: After `buildGrid()`, set `table.style.width = totalW + 'px'` explicitly (42 + 26×82 = 2174px)

## 13. Key Rules & Decisions Made

1. **D: drive** — All file paths default to `D:\JainSheet\`. Open/Save dialogs start there.
2. **Indian locale** — Currency format uses ₹ with Indian numbering (en-IN locale).
3. **No C: drive references** — Project was originally on Desktop (C:), moved to D: drive.
4. **Icon design** — Green rounded square, grid lines, large "J" letter, "SHEET" text at bottom. 7 sizes embedded (16, 24, 32, 48, 64, 128, 256px).
5. **`^latest` is invalid** — Always use real version numbers in package.json devDependencies.
6. **`installerName` is invalid NSIS** — Use `artifactName` instead.
7. **node_modules never committed** — Always in .gitignore.
8. **Two IPC blocks bug** — At one point renderer.js had duplicate IPC handlers causing every action to fire twice. All duplicates removed.
9. **Fill handle rules (must not change):**
   - Drag DOWN → fills with value+1, value+2...
   - Drag RIGHT → fills with value+1, value+2...
   - Drag UP → does nothing
   - Drag LEFT → does nothing
10. **Close window** — Never use IPC round-trip for close. Always handle entirely in main.js with native dialog.
11. **Syntax checking** — Always run `node --check filename.js` before declaring files error-free.
12. **XLSX save** — Uses SheetJS (`xlsx` npm package). Exported as base64, written as binary buffer by main.js.
13. **JSON save format v2** — Includes all sheets array. Backward compatible with v1 single-sheet format.
14. **Formula autocomplete trigger** — Shows on `=` immediately (12 functions), narrows on typing, shows signature on `(`.
15. **Cell click during formula** — Must append ref to formula, NOT commit the edit.
