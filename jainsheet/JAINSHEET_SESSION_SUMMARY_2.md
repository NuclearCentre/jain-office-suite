**Bugs fixed in the xlsx file:**

| # | Bug | Root Cause | Fix Applied |
|---|---|---|---|
| 1 | Preview panel crash: `getElementsByTagName` error | openpyxl writes internal sheet hyperlinks with `TargetMode="External"` — crashes xlsx.js parser | Removed from `.rels` file; switched hyperlinks to use `location="SheetName!A1"` attribute directly in XML |
| 2 | Cells not readable by xlsx.js | openpyxl stores strings as `inlineStr` type — not handled by all parsers | Converted all 207 string cells across 6 sheets to proper shared strings format with `sharedStrings.xml` |
| 3 | Title showing `JainSheet &#8212;` instead of `JainSheet —` | Double-escaped HTML entities in XML (`&amp;#8212;` instead of `—`) | Decoded all numeric XML entities back to real Unicode characters |
| 4 | workbook.xml.rels used absolute paths | openpyxl bug — wrote `/xl/worksheets/sheet1.xml` instead of `worksheets/sheet1.xml` | Patched to relative paths |

**Welcome sheet features added:**
- 5 clickable blue hyperlinks (→ Sales Data, → Formula Reference, etc.) that jump to correct sheet
- JainSheet branding row: `Created with JainSheet v2.0 | 2026 JainSheet | github.com/NuclearCentre/JainSheet`
- Tips row: `Ctrl+S = Save | Ctrl+W = Close Workbook | Alt+F4 = Close App`

**SUM formula verification (all confirmed working in file):**

- Column SUM: `=SUM(E2:E13)` — annual total

## 5. Key Rules & Decisions (carry forward)

1. **D: drive** — All file paths default to `D:\JainSheet\`. Open/Save dialogs start there.
2. **Indian locale** — Currency format uses ₹ with Indian numbering (en-IN locale).
3. **Fill handle rules (must not change):** DOWN = value+1, RIGHT = value+1, UP = nothing, LEFT = nothing.
4. **Close window** — Never use IPC round-trip for close. Always handle in main.js with native dialog.
5. **`^latest` is invalid** — Always use real version numbers in package.json devDependencies.
6. **`installerName` is invalid NSIS** — Use `artifactName` instead.
7. **node_modules never committed** — Always in .gitignore.
8. **Syntax checking** — Always run `node --check filename.js` before declaring files error-free.
9. **GPU Cache warnings** — `ERROR:gpu_disk_cache.cc` etc. in terminal are harmless, ignore them.
10. **GitHub tokens** — Generate fresh token each session, use as password, delete after use. Never save in project files.
11. **Excel file name** — The companion file is `JainSheet.xlsx` (NOT ClaudeSheet.xlsx).
12. **openpyxl hyperlink bug** — openpyxl always writes internal links as `TargetMode="External"`. Must patch the ZIP XML directly after saving to fix. Use `location=` attribute, not `r:id=`.
