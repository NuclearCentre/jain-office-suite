### 3.2 XLSX Round-Trip (Save + Open)

**Problem:** Saving as .xlsx lost all formulas and formatting. Opening .xlsx crashed.

**Fixes in main.js:**
- Reading `.xlsx`/`.xls` files now uses `fs.readFileSync().toString('base64')` instead of `utf8`
  (binary files get corrupted if read as text)

**Fixes in renderer.js -- exportXlsx():**
- Rewrote from `aoa_to_sheet` (array-of-arrays, loses everything) to cell-by-cell construction
- Now writes `cell.f` with the formula string so Excel recalculates it
- Now writes `cell.v` with the last-computed value so file opens correctly before recalc
- Now writes `cell.s` style object with: font (bold/italic/underline/strike/size/family/color),
  fill (background color), alignment (h-align + wrap), borders (all 4 sides with style+color),
  number formats (currency/USD/percent/decimal/integer/scientific/date/time)
- Now writes `ws['!merges']` for merged cell regions
- Now writes `ws['!cols']` for column widths
- Uses `XLSX.write()` with `{cellStyles: true}`

**Added importXlsx() in renderer.js:**
- Uses `XLSX.read()` with `{cellStyles:true, cellFormula:true, cellDates:true, cellNF:true}`
- Reads formulas (`cell.f`), values, number formats, full styling
- Reads borders (converts SheetJS border objects to CSS strings)
- Reads column widths (converts Excel `wch` units to pixels)
- Reads `ws['!merges']` and builds `mergedCells` object
- Handles all sheets in the workbook

**Updated menu-open handler:**
- Now routes `.xlsx`/`.xls` to `importXlsx()` instead of `loadState()` (which would crash)

### 3.3 Merge & Center (Full Implementation)

Previously was an alert() stub. Now fully working.

**Data model:**
- `mergedCells` object per sheet, key = master cell ID (e.g. "A1")
- Value = `{r1, c1, r2, c2}` defining the full merged rectangle
- Stored in all sheet state: `sheets[]` array, `getState()`, `loadState()`

**Functions added:**
- `isMergeHidden(r, c)` -- returns true if cell is a slave inside a merge
- `getMasterCell(r, c)` -- returns `{r, c}` of master if cell is inside any merge
- `mergeToggle()` -- merge selection if not merged; unmerge if master clicked again
- `unmergeRange(masterId)` -- removes merge, restores individual cells
- Merge and unmerge both push to undoStack

**renderCell() updated:**
- Checks `isMergeHidden()` first -- slave cells get `display:none`
- Master cells get `colSpan` and `rowSpan` applied to `<td>`
- Master defaults to `text-align:center; vertical-align:middle`

**Click handling:**
- Clicking a slave cell redirects to master via `getMasterCell()` in `startSel`

**Save/Load:**
- `getState()` includes `mergedCells` in v2 JSON
- `loadState()` restores `mergedCells` per sheet
- All `sheets.push()` calls now include `mergedCells:{}`

**XLSX:**
- `exportXlsx()` writes `ws['!merges']` array
- `importXlsx()` reads `ws['!merges']` and rebuilds `mergedCells`

### 3.4 Color Picker (Full Implementation)

Previously was a `prompt()` dialog. Now a proper popup.

**How it works:**
- Click Font Color or Fill Color button in ribbon -> popup appears below the button
- 50 preset colors in a 10x5 grid (dark grays, reds/oranges, greens, blues, purples)
- Swatches scale up on hover for visibility
- Native `<input type="color">` for unlimited custom colors
- "None" button to remove the color
- Closes on click outside the picker
- Positions below button, flips upward if near bottom of screen

**Functions:**
- `openFontColor(btnEl)` / `openFillColor(btnEl)` -- buttons pass `this`
- `showColorPicker(anchorEl, target)` -- builds and shows the popup
- `applyColorPickerValue(hex)` -- applies color and closes picker
- `closeColorPicker()` -- removes popup, removes event listener
- `_colorPickerOutside(e)` -- outside-click handler

**index.html changes:**
- Font Color and Fill Color buttons updated to pass `this`:
  `onclick="openFontColor(this)"` and `onclick="openFillColor(this)"`
